# Evans

An active Lean 4 reference project and source-based blueprint following
Lawrence C. Evans, *Partial Differential Equations* (AMS GSM 19).

The blueprint covers the full text. Lean currently covers Chapter 1 and the
transport equation, with Laplace, heat, and wave equations under development.
Blueprint sources live in `blueprint/src/`, with chapters under
`blueprint/src/chapters/`.
